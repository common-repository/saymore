An opt-in plugin for collecting email addresses of those who comment on your blog.

== Description ==

SayMore is a email opt-in plugin that adds users to an ExactTarget list using the web collect method once they comment on a post/page in your blog.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
